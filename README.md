# Running

npm run dev

heroku local web --port=3000

## Setup

Create a .env file with this structure:

```
SHOPIFY_API_KEY=KEY
SHOPIFY_API_SECRET_KEY=SECRET
HOST='HTTPS_URL'
API_VERSION='YEAR-MONTH'
```
